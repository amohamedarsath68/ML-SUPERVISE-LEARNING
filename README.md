# 🤖 Machine Learning Projects Collection


## 📌 About

This repository contains Machine Learning projects developed using Python and Scikit-Learn. Each notebook demonstrates a complete workflow including data preprocessing, exploratory data analysis (EDA), model training, evaluation, and prediction.

The projects cover both **Regression** and **Classification** problems using real-world datasets.
