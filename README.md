# 🚗 Car Data Visualization

A simple data visualization project created using Python to explore and analyze a car dataset. 
This project includes data cleaning, exploratory data analysis (EDA), and various visualizations to understand patterns and insights from the data.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Features
- Data Cleaning
- Data Visualization
- Exploratory Data Analysis (EDA)
